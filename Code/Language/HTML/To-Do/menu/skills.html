0.2 Alegra & Infinity

In an algebra or calculus class, you might have found a particular set of numbers (maybe the set of numbers in the range of a function).

You would represent this set as an interval:
[0,infinity) is the range of function(x) = x 2 since the set of outputs of the function are all real numbers 0 and greater.

This set of numbers is NOT discrete.

The numbers in the set are not separated by much at all. In fact, take any two numbers in the set and there are infinitely many more between them which are also in the set.

Discrete math could still ask about the range of a function, but the set would not be an interval. Consider the function which gives the number of children of each person reading this. What is the range?

I'm guessing it is something like . Maybe 4 is in there too. But certainly there is nobody reading this that has 1.32419 children.

This output set is discrete because the elements are separate. The inputs to the function also form a discrete set because each input is an individual person.
